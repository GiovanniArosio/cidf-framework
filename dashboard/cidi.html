Component 03 · Synthesis Layer

CIDICyber-Interpretive Disruption Index — scenario-based

CIDI is a scenario-based exploratory synthesis, not a primary inferential result. It summarizes alternative, transparent modelling choices after the substantive comparison of evidence-aware TCI and the individual IVA components. No unique empirically validated weighting is claimed, and no single value is presented as "the" CIDI.

01

Scenario matrix

CORE & EXTENDED × 3 WEIGHTS

Each level is evaluated under three transparent weight scenarios — neutral (0.50/0.50), interpretive-prioritized (0.40/0.60) and technical-prioritized (0.60/0.40). Core uses the equal-weight Interpretive Vector and is available for all cases. Extended adds the Response Timing Proxy at a lower weight — and is therefore unavailable for KA-SAT, where timing cannot be measured. Cell intensity encodes magnitude; the dot marks the top case per column.

02

What drives each score

TECHNICAL vs INTERPRETIVE
02.1

Composition · interpretive scenario

CIDI Core (interpretive) = 0.40 × TCI + 0.60 × IVC. The split shows how much of each case's score is technical versus interpretive. For PAP the interpretive share exceeds the technical one — the inversion that defines the interpretive void.
02.2

Weight sensitivity · Core

How each case's Core score moves across the three weight scenarios. The bands are narrow and never cross — the substantive comparison is robust to the weighting choice.
03

Ranking stability

ROBUSTNESS, NOT POINT ESTIMATE

The reported result is a ranking-stability comparison, not a point estimate. Across all three Core weight scenarios the ordering is identical, and it holds for the Extended scenarios among the cases where it is available. This is a descriptive ordering over the available cases — no causal or out-of-sample claim is made.

Guarantees & limitations The Technical–Public Gap diagnostic is excluded from every aggregate and never enters a CIDI formula. The Response Timing Proxy is unavailable for KA-SAT and never imputed, so KA-SAT has no Extended CIDI — a valid data limitation, not a missing value to fill. Evidence coverage is shown alongside scores but never mathematically folded into them. All outputs remain exploratory, corpus-bound and non-causal; the corpus is ~15 curated source-derived summaries per case, not the full public sphere.
