Framework Architecture

CORPUS → ANALYSIS → SYNTHESIS
TCI
Technical Complexity Index
Evidence-aware MITRE ATT&CK scoring across five components; reports a floor, an evidence-adjusted score, and evidence coverage.

Interpretive Void Analyzer

CIDI
Cyber-Interpretive Disruption Index
Scenario-based exploratory synthesis combining evidence-adjusted TCI with the Core Interpretive Vector. Not a single point estimate.
02

Composite Index — CIDI

CORE · INTERPRETIVE-PRIORITIZED SCENARIO
Composite Index — CIDI Core
CIDI = 0.40 × TCIEA + 0.60 × IVC
TCIEA = evidence-adjusted mean of documented ATT&CK components
IVC = 0.5 × Attribution_Drift + 0.5 × Narrative_Fragmentation
Asymmetric weighting reflects the primacy of communicative disruption over technical sophistication. This is the interpretive-prioritized scenario of CIDI Core — one of three transparent weight scenarios (neutral 0.5/0.5, interpretive 0.4/0.6, technical 0.6/0.4); no single weighting is claimed as definitive. Values are exploratory and corpus-bound.

IVA Dimensions

FOUR INSTRUMENTS · REPORTED INDIVIDUALLY
Attribution
Drift
Instability of responsibility claims over time, from a manual, text-grounded structured coding protocol — no keyword or LLM extraction.
CORE
Narrative
Fragmentation
Diversity of competing narratives via K-Means clustering on sentence embeddings, reported with k = 3 / 4 / 5 sensitivity. Exploratory.
CORE
Technical–
Public Gap
Semantic, lexical and length distance between technical and public summaries. Excluded from every aggregate and every CIDI formula.
DIAGNOSTIC
Response
Timing Proxy
Duration of the communicative vacuum before authoritative response. Captures temporal pressure; returns an explicit unavailable state when evidence is thin.
